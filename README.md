<h1 align="center" style="padding:0;margin:0;border:none;">Build</h1>

## Installation
```javascript
npm install -g vue-build
```

## Scripts
```javascript
// Initiate files/folders
vue-build init

// Run dev server
vue-build dev

// Build
vue-build build

// Unit testing
vue-build unit

// E2e testing
vue-build e2e

// Linting
vue-build lint
```
